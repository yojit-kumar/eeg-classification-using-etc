# Data

This folder is intended to store the dataset required for reproducing the results described in the research paper. Follow the steps below to download and organize the data:

## Instructions for Downloading the Dataset

1. **Download the Dataset**  
   Please download the dataset from the following source:  
   - [EEG Motor Movement/Imagery Dataset](https://physionet.org/content/eegmmidb/1.0.0/)

2. **Organize the Data**  
   After downloading, extract the dataset into this `data/files/` folder. The folder structure should look like this:
   ```
   |-- data/
       |-- files/          # All dataset files
   ```

3. **Verify the Data**  
   Ensure that all the downloaded data files are correctly placed in the `data/files/` folder before running any scripts.

## Notes

- Make sure you have sufficient storage space before downloading the dataset.
- If any preprocessing is required, it will be handled by the scripts in the `notebooks/` or `src/` folder.
- If you encounter any issues with downloading or organizing the dataset, please contact the repository owner.
